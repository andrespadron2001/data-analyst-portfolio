# Portafolio | Analista de Datos

Analista de Datos orientado a negocio, especializado en transformar información operativa, comercial y financiera en insumos claros para la toma de decisiones.

Este portafolio reúne casos prácticos que simulan escenarios reales de empresa, donde el análisis de datos se utiliza para identificar riesgos, detectar oportunidades y proponer acciones concretas.

## Enfoque
- Análisis orientado a decisiones, no solo a reportes
- Visualización ejecutiva y KPIs accionables
- Automatización de procesos de datos
- Estandarización y control de información

## Herramientas
- Excel avanzado (modelado, validación y control)
- Power BI (modelado, DAX y dashboards ejecutivos)
- Python (pandas, automatización y ETL básico)
- SQL (extracción y consultas)
- Git / GitHub (control de versiones)

## Casos de negocio
- **01 Ventas**: Evaluación de desempeño comercial y priorización
- **02 Marketing**: Análisis de funnel y eficiencia de inversión
- **03 Finanzas**: Control financiero y proyección de flujo
- **04 Operaciones**: Capacidad instalada y eficiencia operativa
- **05 Automatización**: Reducción de trabajo manual mediante scripts
